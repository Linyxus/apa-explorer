# APA Explorer

An interactive explorer of sessions and tasks for "Agentic Proof Automation: A Case Study". It contains human-agent interactions for mechanizing System Capless. [Source Code for Mechanization](https://github.com/linyxus/semantic/tree/case-study)

Access the explorer at https://apa.univalence.xyz.

## Deploy Locally with Docker

1. **Start the application**

   ```bash
   docker compose up --build
   ```

2. **Open in browser**

   Visit http://localhost:22025
